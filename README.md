# 图像处理/人像美化引擎PC版

#### 介绍
zphotoengine升级版，包含照片美化和人脸美化两部分，使用非常简单，直接DLL包调用即可

#### 使用说明
imageengine_libs文件夹为DLL调用的库文件，其中
1. inc包含了c头文件
2. lib包含了dll库文件
3. resource包含了人脸美化库用到的资源文件
